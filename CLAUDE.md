## Opus Finance is a personal finance tracker created for the web using React. This app aims to provide a suite of informative, intuitive, and useful finance tools with Plaid integration, graphs, and lists. It will have 3 major personal finance features:

1. **Tracking:** The ability to sync all of someones accounts and transactions into one app via Plaid, including:
    - Net worth tracking
    - Custom reports
    - Credit score tracking

2. **Budgeting:** An assortment of budgeting tools such as:
    - A flex budget system
    - Custom categories

3. **Planning:** The ability to track goals and make progress towards completion. Features include:
    - Savings goals
    - Debt paydown goals
    - Goal forecasting

See /docs/wireframes for the reference images that should guide development by serving as examples of how the pages, tabs, and features should appear.
